# eslint-config-monkeyfe

> eslint Config  for monkey fe  team.s

### 安装
```sh
npm install --save-dev  eslint-config-monkeyfe eslint
```

Then add following contents to your .eslintrc file
```
{
  "extends": "monkeyfe"
}
```
